Aplicatia va fi folosita pentru un birou de inchirieri auto, aceasta va afisa o lista cu toate masinile firmei. Pentru a avea acces la 
aplicație, vor fi folosite datele de conectare puse la dispoziție de firmă pentru fiecare angajat. În momentul în care un client este interesat, vom putea vizualiza lista mașinilor disponibile.
Închirierea se face pe baza de nume, prenume, CNP si  pentru o anumita perioada de timp, toate acestea fiind salvate în baza de date, în același timp mașina devenind indisponibilă pentru următorii clienți.
